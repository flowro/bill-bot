# Bill Bot 🧾

Telegram bot that tracks bills and expenses for tradespeople. Snap a photo, forward a receipt, ask questions about your spending.

**Built for plumbers, builders, and small business owners who live on their phone.**

## The Problem

Tradespeople manage expenses on paper, spreadsheets, or not at all. They're on job sites all day — they don't have time for accounting software. But they need to track costs for tax, invoicing clients, and knowing if they're actually making money.

## The Solution

A Telegram bot that speaks their language:

- 📸 **Snap a receipt** → auto-categorizes (materials, fuel, tools, food)
- 💬 **"How much did I spend on materials this month?"** → instant answer
- 📊 **Weekly/monthly summaries** → sent automatically
- 🏷️ **Tag by job/client** → "This one's for the Johnson bathroom"
- 📤 **Export** → CSV/PDF for accountant at tax time

## Beta

Testing with family first — brother-in-law is a plumber. Real-world validation before scaling.

## Stack (planned)

- Telegram Bot API
- Claude Vision (receipt OCR + categorization)
- Supabase (storage + auth)
- Node.js
